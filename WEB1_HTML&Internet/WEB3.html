JavaScript

JavaScript, 흔히 JS로 줄여서 불리는 이 언어는 HTML과 CSS와 함께 웹의 핵심 기술인 프로그래밍 언어입니다. 99%의 웹사이트가 클라이언트 측 웹페이지 동작을 위해 JavaScript를 사용합니다. 웹 브라우저에는 클라이언트 코드를 실행하는 전용 JavaScript 엔진이 있습니다. 이러한 엔진은 일부 서버와 다양한 애플리케이션에서도 사용됩니다. 비브라우저 용도로 가장 인기 있는 런타임 시스템은 Node.js입니다. JavaScript는 ECMAScript 표준을 준수하는 고수준, 종종 즉시 컴파일되는 언어입니다. 동적 타이핑, 프로토타입 기반 객체 지향, 일급 함수 기능을 가지고 있습니다. 이벤트 주도, 함수형, 명령형 프로그래밍 스타일을 지원하는 다중 패러다임 언어입니다. 텍스트, 날짜, 정규 표현식, 표준 데이터 구조 및 문서 객체 모델(DOM)을 다루기 위한 응용 프로그래밍 인터페이스(API)를 제공합니다.
