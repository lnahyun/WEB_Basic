HTML

Hypertext Markup Language (HTML)은 웹 페이지와 웹 애플리케이션 을 만드는 데 사용되는 표준 마크업 언어입니다. Cascading Style Sheets (CSS)와 함께 사용됩니다. 웹 브라우저는 웹 서버나 로컬 저장소에서 HTML 문서를 받아 멀티미디어 웹 페이지로 렌더링합니다. HTML은 웹 페이지의 구조를 의미론적으로 설명하며, 원래 문서의 외형에 대한 단서를 포함했습니다.

HTML 요소는 HTML 페이지의 구성 요소입니다. HTML 구성 요소를 사용하여 이미지 및 대화형 양식과 같은 기타 객체를 렌더링된 페이지에 포함시킬 수 있습니다. HTML은 제목, 단락, 목록, 링크, 인용문 및 기타 항목과 같은 텍스트에 대한 구조적 의미를 나타내어 구조화된 문서를 생성하는 수단을 제공합니다. HTML 요소는 태그로 구분되며, 각 태그는 꺾쇠 괄호를 사용하여 작성됩니다.
