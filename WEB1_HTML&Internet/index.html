WEB

Web(웹)은 인터넷을 통해 문서와 기타 리소스를 상호 연결된 하이퍼텍스트로 제공하는 시스템입니다. 웹은 월드 와이드 웹(World Wide Web, WWW)이라고도 불리며, 주로 웹 브라우저를 통해 접근합니다. 웹은 HTML, CSS, JavaScript와 같은 기술을 기반으로 동작하며, 이들 기술을 통해 웹 페이지의 구조, 스타일, 그리고 상호작용을 정의합니다. HTML은 웹 페이지의 콘텐츠와 구조를 정의하고, CSS는 그 스타일과 레이아웃을 지정하며, JavaScript는 동적인 동작과 사용자 인터랙션을 구현합니다. 웹 브라우저는 사용자가 URL을 입력하면 해당 주소의 서버에 요청을 보내고, 서버는 HTML 문서와 관련 리소스를 응답으로 반환합니다. 브라우저는 이 문서를 해석하고 렌더링하여 사용자가 볼 수 있는 웹 페이지를 만듭니다. 이 과정에서 CSS 파일을 불러와 스타일을 적용하고, JavaScript 코드를 실행하여 페이지의 동작을 제어합니다. 이러한 방식으로 웹은 사용자에게 동적이고 인터랙티브한 경험을 제공합니다. 오늘날 대부분의 웹사이트는 이 세 가지 기술을 결합하여 복잡하고 풍부한 사용자 인터페이스를 제공합니다.
